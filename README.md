# Smart Academic Management System

This is a full-stack Django web application designed to help students manage their course enrollments. It features a secure login system and a personal dashboard where students can view, add, and drop their classes.

## Features
- **Secure Authentication**: User registration and login/logout functionality.
- **Student Dashboard**: A personalized view of enrolled courses.
- **Full CRUD Functionality**: 
    - **Create**: Enroll in new courses.
    - **Read**: View current course list.
    - **Update**: Edit registration details.
    - **Delete**: Drop classes from the schedule.
- **Responsive UI**: Built using Django Template Inheritance for a consistent look.

## Tech Stack
- **Backend**: Python 3, Django 5.0
- **Database**: SQLite3
- **Deployment**: Hosted on PythonAnywhere
